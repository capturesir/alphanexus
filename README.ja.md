# AlphaNexus — ポートフォリオ管理プラットフォーム

[中文](README.md) | [简体中文](README.zh-CN.md) | [English](README.en.md) | **日本語**

> 🌐 **Live Demo**: [https://www.alphanexus.cc](https://www.alphanexus.cc)

AlphaNexusは**モバイルファースト**の多言語ポートフォリオ追跡プラットフォームです。**単一HTMLフロントエンド + ゼロ依存Node.jsバックエンド**で構成され、クロスマーケット・多通貨の正確な会計処理とパフォーマンス分析に特化しています。最小構成のVPSで稼働し、`file://`でオフライン利用も可能です。

> ⚠️ **免責事項**: 本プラットフォームは個人の資産管理ツールです。市場データ、為替レート、ニュースはすべて第三者ソースからの情報を参考として提供するものであり、**いかなる投資助言を構成するものでもありません**。ニュースは見出しインデックスとして表示され、元の出版社へのリンクを含みます。著作権は各出版社に帰属します。

## システム画面

👉 [Live Demo で体験](https://www.alphanexus.cc) — ゲストモードで登録なしに利用可能。

## 主な特長

- **モバイルファースト設計**: タッチターゲット44dp以上、ジェスチャーチャート、下部タブナビゲーション。タブレット・デスクトップにも対応。
- **多言語対応**: 繁體中文、简体中文、English、日本語に即時切り替え可能（外部JSON言語パック＋オフライン用内蔵フォールバック）。
- **多市場・多通貨**: 米国株/ETF、香港株、A株、日本株、暗号資産。多通貨の残高は履歴為替レートで基準通貨に自動換算。
- **業界標準の会計モデル**: 原価評価＋取引単位数リステイトメント＋内部資金移動により、配当、株式分割、手数料を正確に処理し、純資産価値の歪みや二重計上を防止。
- **レジリエントなデータレイヤー**: 複数ソースのフォールバック、ローカル永続化＋増分更新、引け後の自動プリフェッチ、リクエストコアレッシング、gzip圧縮。
- **本格的なアカウントシステム**: scryptパスワードハッシュ、Bearerトークン、任意のメール認証、ポートフォリオのクラウド同期。
- **プライバシーと共有**: ワンタップでプライバシーモードを切り替え、金額を非表示。金額を含まないリターンのみの成績カードを共有可能。
- **ゼロ依存**: バックエンドはNode.js組み込みモジュールのみ使用。`npm install`は不要。

## 機能一覧

### 資産リスト
- ポジション一覧: 時価評価額、損益、リターン率、受取配当、過去12ヶ月の配当見込み。
- **トータルカード**: 金額タップでプライバシーモード切替。損益タップで「累計損益/直近2営業日の変動」を循環切替。右上の「分析」で分析ページへ。
- **ポジションソート**: 時価評価額、日次変動率%、リターン率（昇順/降順）。
- **金額欄のタップ循環**: 最新時価評価額 → 最新株価（日次%）→ 日次時価変動額。
- **ポジション詳細ページ**: ポジションをタップすると最新株価、ミニスパークライン、当該資産の全取引履歴、配当、データソースを表示。資産全体を削除可能。
- **多通貨カッシュボード**: 各通貨の残高（マイナス/信用取引対応）、累計配当・利息収入、累計手数料・負債支出。
- **取引履歴**: 9種類の取引タイプ（株式/債券/現金/オプション/その他/配当/利息/手数料/負債）。手数料フィールド付き。
- **取引フィルター**: グループ化ドロップダウン（取引タイプ/銘柄コード）。銘柄コードでのフィルタリングは関連する配当も含む。
- **資産ロケーション**: 証券会社/銀行を区別（例: Futu、Bank of China）。アイコン付きピッカーで既存ロケーションを選択。空白の場合は「デフォルト」。

### ポートフォリオ分析
- **資産配分**: 現金/株式/債券/その他の円グラフ。比率と金額を表示。
- **パフォーマンス概要**: ポートフォリオのTWRと主要指数（SPY、VT）の同期間比較。
- **著名投資家の追跡**: 米国SEC EDGAR 13F公開情報に基づき、13名の著名投資家の直近四半期のトップ10保有銘柄（名称/金額/比率）を表示。対象: Buffett (Berkshire)、Li Lu (Himalaya)、Tepper (Appaloosa)、Ackman (Pershing)、Druckenmiller (Duquesne)、Burry (Scion)、Griffin (Citadel)、Cohen (Point72)、Coleman (Tiger Global)、Wood (ARK)、**Dalio (Bridgewater)**、**Loeb (Third Point)**、**Klarman (Baupost)**。OpenFIGI APIによりCUSIPからティッカーを自動解決（最大100%カバレッジ）。米国ロングポジションのみ、四半期末から最大45日の遅延、現金を含まないなどの制限事項を明記。
- **配当カレンダー**: 月次配当見込み（過去12ヶ月の実績に基づく）。権利落ちタイムライン（上方向: 過去1年の実際のコーポレートアクション、下方向: 来年の想定権利落ち日）。

### パフォーマンスチャート
- 3種類の指標: NAV、TWR（時間加重）、MWR（資金加重）。TWRとMWRは累積ベースで一致し、外部資金フローがない場合は同一値となる。
- 4種類のチャートタイプ: ライン、スムーズカーブ、ローソク足（日次/週次/月次/年次集計）。
- **ジェスチャー操作**: ピンチズーム、シングルフィンガーパン、長押しスクラブ（日付＋ポートフォリオ＋各比較シリーズの数値を表示）、指を離した後も読み取り値を維持。
- **ベンチマーク比較**: ETF/株式（例: VT、SPY）をトータルリターン（調整後価格）カーブとしてオーバーレイ表示。
- **著名投資家の仮想カーブ**: 投資家を選択すると、四半期13F保有銘柄に基づくシミュレーションNAVカーブをオーバーレイ。免責エリアに計算方法の説明とSECファイリングリンク付きの最新保有銘柄詳細を表示。
- **イベントマーカー**: 権利落ち（💰）と株式分割（✂️）をX軸にアイコン表示。十字線で詳細を表示。
- **期間切替**: 1M / 3M / 6M / YTD / 1Y / 3Y / ALL。資産ロケーションでフィルタリング可能。
- **インタラクティブチャート**: ポジション横のチャートアイコンをタップして起動。ローソク足チャート＋12種類のテクニカル指標に対応。
  - 指標ツールボックスパネル: オン/オフ切替とパラメータ設定。オーバーレイ（SMAなど）と独立パネル（RSIなど）をサポート。
  - テクニカル指標: SMA、EMA、RSI、MACD、ボリンジャーバンド、ストキャスティクス（KD）、VWAP、ATR、OBV、ADX、Parabolic SAR、一目均衡表。
  - 十字線ホバー時、すべての指標パネルが同日の値を同期表示。
  - 指標データの表示範囲はメインチャートの表示範囲に連動。
- オプションはカーブから除外。チャートに「オプション時価 ±X 除外」と注記。

### ニュース
- 3つのタブ: 保有銘柄関連 / トレンド / 全体。
- 企業名またはファンド名で関連ニュースをスマート検索（非米国銘柄に最適化）。
- **見出し、ソース、時刻のみを表示し、元記事へリンク**。コンテンツの転載は行わない。

### プロフィール
- アバターとプロフィール、パスワード変更、ログアウト、**アカウント削除**（パスワード確認後、すべてのデータを完全削除。GDPR/PDPOの削除権に対応）。
- ダーク/ライトテーマ、言語切替、騰落色設定、デフォルトチャートタイプ。
- **プライバシーモード**（デフォルトON）: すべての金額を非表示（•••と表示）。比率と株価は通常通り表示。
- **損益表示**のデフォルトモード（累計損益 / 日次変動）。
- **リターン共有**: 金額を含まないリターンのみの成績カードを生成。CanvasでPNGに書き出し、システム共有パネルで保存または直接ダウンロード。
- **自動配当**の切替と**配当源泉徴収税率**の設定。
- **カスタムデータソース**（JSONPath）、**CSVインポート/エクスポート**、デモポートフォリオの読み込み/クリア。

## 会計モデル（原価 + 内部資金移動）

- **評価方法**: 個人ポートフォリオは常に原価（株式分割調整済み、配当未調整）で評価。ベンチマーク比較のみ調整後価格（トータルリターン）を使用。
- **配当**: 権利落ち日に各資産ロケーションの保有株数に応じて自動仕訳。税引後金額 = 保有株数 × 1株当たり配当 × (1 − 税率)。自動仕訳は⚡マーク付き。編集すると手動仕訳に変更、削除するとスキップリストに追加。同一銘柄は±14日で重複排除。権利落ち日の株価下落と現金増加によりNAVは平滑化され、外部資金フローとして計上されない。
- **株式分割**: 取引単位数をリステイトメント（単位数 × F、単価 ÷ F）。ユーザーの記録は書き換えられず、NAVやコストにジャンプが生じない。
- **オプション**: カーブおよびTWR/MWRから除外。現金の出し入れは外部フローとして処理（完全中立）。リストでは最終清算価格で静的評価。
- **手数料 / 負債**: 内部支出。現金を減少させるがフローには計上せず、自然にリターン率を低減。
- **TWR**: `∏ Vₜ/(Vₜ₋₁+Cₜ) − 1`（手数料の二重計上なし）。**MWR**: XIRR、累積ベースと年率ベースの2種類。

## データアーキテクチャ

- **価格データのフォールバックチェーン**: Yahoo Finance → 期限切れローカルストア → Stooq（米国/香港/日本）。暗号資産: Yahoo → CoinGecko。
- **為替レートのフォールバック**: Yahoo `{CCY}=X` → ローカルストア → Frankfurter / ECB。クロスレートはUSD経由の三角換算。
- **永続化 + 増分更新**: 履歴データは`data/market/`に保存。30分以内ならストアから応答、超過分のみ取得。新しい株式分割イベント検出時のみ全量再取得。
- **引け後プリフェッチ**: 毎日台北時間05:00（`PREFETCH_HOUR`で変更可能）にすべての保有銘柄とカスタムソースを自動更新。
- **カスタムデータソース**: JSONPathで任意のAPIを接続（日付フォーマット自動検出、SSRF対策、連続失敗アラート）。
- **著名投資家のCUSIP自動解決**: SEC 13FのCUSIPをOpenFIGI APIでティッカーに自動変換。結果は`data/cusip_cache.json`にキャッシュ（メンテナンス不要、自動蓄積）。
- **プラガブルニュースレイヤー（市場別自動ルーティング）**: **香港株（.HK）、A株（.SS/.SZ）、台湾株（.TW）は中国語でGoogle Newsを自動検索、日本株（.T）は日本語で検索、米国株はYahoo（英語）を維持**。すべて企業名で検索。設定不要で即座に動作。出典を保持しコンテンツは転載しない。`NEWS_PROVIDER=rss`で全銘柄をRSSに強制切替、または`NEWS_PROVIDER=newsapi|marketaux` + `NEWS_API_KEY`でライセンス付きソースに切替可能。ニュースは**銘柄単位**でキャッシュおよびリクエストコアレッシング。各銘柄はサイクルごとに最大1回のみ取得（ユーザー数に依存しない）。


## クイックスタート

**Node.js 18以上**が必要（組み込み`fetch`を使用）。`npm install`は不要。

```bash
node server.js            # デフォルト http://localhost:8080
PORT=3000 node server.js  # ポート変更
```

ブラウザで`http://localhost:8080`を開きます。初回起動時は「ゲストモード」を選択し、デモポートフォリオを読み込んで即座に体験できます。

### 著名投資家データのビルド

著名投資家機能の初回使用前に、サーバーでビルドを実行してください（SEC 13Fデータ＋履歴株価を取得）。

```bash
node server.js --build-gurus   # すべての著名投資家をビルド（約10〜15分）
```

### 環境変数

| 変数 | 説明 | デフォルト |
|---|---|---|
| `PORT` | サーバーポート | 8080 |
| `PREFETCH_HOUR` | 毎日のプリフェッチ時刻（ローカル時間、-1で無効化） | 5 |
| `SMTP_HOST` / `SMTP_PORT` / `SMTP_USER` / `SMTP_PASS` / `SMTP_FROM` | メール認証付きサインアップを有効化（465は暗黙的TLS） | 未設定 = 認証なし |
| `NEWS_PROVIDER` | ニュースソース: `rss`（Google News RSS、中国市場）/ `newsapi` / `marketaux` | 未設定 = Yahoo集約 |
| `NEWS_API_KEY` | newsapi / marketaux用のキー（rssには不要） | なし |
| `MARKET_FRESH_MS` | 市場データの鮮度ウィンドウ | 30分 |
| `SEC_UA` | 著名投資家の13F検索用SEC User-Agent（実際のメールアドレス推奨） | 組み込みプレースホルダー |

## デプロイメント

VPS環境から完全に構築するガイド（リバースプロキシ＋自動HTTPS＋バックアップ＋Cloudflare）: **[DEPLOY.md](DEPLOY.md)** を参照。

最小構成のVPS（1 vCPU / 1GB RAM、月額約US$4〜6）で十分に動作します。DDoS対策としてCloudflare無料プランの導入を推奨。アプリケーション層でもIP単位のレート制限と低速リクエストタイムアウトを実装済み。

## 容量とスケール

- **個人利用**: 月間トラフィック約30〜100MB、ストレージ約2〜3MB。時間経過とともに無制限に増大しない。
- **10,000ユーザー**: 総ストレージ約3GB以内。上流APIへのリクエスト数はユーザー数ではなくユニーク銘柄数に比例。永続化、プリフェッチ、コアレッシングによりデータソースの許容範囲を十分に下回る。
- ユーザー/セッション格納が約1,000〜2,000人を超える場合、SQLite（Node 22+に組み込み）への移行を推奨。

## 法務とコンプライアンス

- 市場データとニュースは第三者ソースから取得し、**個人/デモ用途**を想定しています。**商用展開前に**、適切にライセンスされたデータおよびニュースAPIへの切り替えと、現地弁護士への相談を行ってください。
- ニュースは「見出し＋ソース＋リンク」のインデックスとして元ソースへリダイレクトし、**コンテンツの転載は行いません**。著作権は各出版社に帰属します。

## 技術スタック

- フロントエンド: バニラHTML/CSS/JavaScript、Canvas描画チャート。フレームワークなし、ビルドステップなし。
