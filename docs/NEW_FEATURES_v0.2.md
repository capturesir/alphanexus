## 功能一：股息行事曆（Dividend Calendar & Forecast）

### 1.1 概述

在資訊頁新增「💰 股息」分頁，提供：
- **月度長條圖**：未來 12 個月預估現金股息收入
- **時間軸行事曆**：未來 30 天的除息日（Ex-dividend）和派息日（Payment Date）

### 1.2 決策記錄

| 項目 | 決策 |
|---|---|
| 港股/日股數據不全 | 全部顯示，但標註「數據可能不全」 |
| 股息口徑 | Yahoo 過去 12 個月實際派息推算 |
| 位置 | 資訊頁新分頁，與「持倉相關/熱門/全部」並列 |

### 1.3 數據來源

| 數據 | 來源 | 現狀 |
|---|---|---|
| 歷史股息 | `/api/history` 的 `dividends` 欄位 | ✅ 已有 |
| 未來除息日 | Yahoo `calendarEvents` | ⚠️ 需驗證可用性 |
| 年化股息率 | Yahoo `trailingAnnualDividendYield` | ✅ 可用 |
| 持股數 | `state.txns` 計算 | ✅ 已有 |

### 1.4 技術架構

```
後端新增:
  /api/div-calendar
    輸入: 用戶持倉代碼列表
    輸出: {
      monthly: [{month:"2026-07", total:1234.56, items:[{sym,divPerShare,shares,amount}]}],
      upcoming: [{sym, exDate, payDate, divPerShare, estAmount}]
    }
    邏輯:
      1. 遍歷持倉代碼
      2. 讀取 data/market/ 的歷史股息
      3. 計算過去 12 個月每股總股息 ÷ 12 = 月均每股股息
      4. × 持股數 = 月度預估
      5. 讀取 calendarEvents 取得未來除息/派息日

前端:
  資訊頁 → 新增「💰 股息」分頁
  - 月度長條圖: 12 格 SVG bar,hover 顯示明細
  - 時間軸: 卡片列表,顯示代碼/日期/預估金額
  - 標註: 「預估僅供參考，實際派息以公司公告為準」
  - 數據不全提示: 港股/日股顯示「部分市場數據可能不全」
```

### 1.5 UI 設計

- **分頁標籤**：pill 樣式 + 💰 圖示，色調偏暖黃（`#f59e0b`）
- **月度長條圖**：
  - 12 格，每格顯示月份縮寫 + 預估金額
  - 顏色：綠色系（收入）
  - Hover/點擊：展開明細（代碼 × 每股股息 × 持股數）
- **時間軸**：
  - 卡片式列表，按日期排序
  - 每張：代碼 | 除息日 | 派息日 | 預估金額
  - 無數據的代碼顯示「暫無未來派息資訊」
- **免責提示**：頁面底部小字

|---|---|
| 名單 | 直接擴充到 10 位 |
| 高周轉/多空型 | 收錄但加失真警告 |
| 負現金利率 | 3% 年利率（每日計息） |
| 回溯長度 | 3 年（12 個季度） |
| 初始化方式 | 部署後手動跑一次腳本 + 之後排程更新 |

### 2.5 虛擬淨值計算邏輯

```
初始化(3年前的第一個交易日):
  初始資金 = $1,000,000
  讀取最早的 13F 季度持倉
  按持倉比例買入(用當日收盤價計算股數)
  剩餘 = 虛擬現金

季度調整日(13F 公布後):
  1. 計算當前持倉市值 + 現金 = 總資產 A
  2. 讀取新季度 13F 持倉
  3. 按新比例重新分配:
     - 需要買入 → 從現金扣
     - 需要賣出 → 現金增加
     - 現金不足 → 允許為負(外部資金注入)
  4. 負現金: 每日利息 = |負現金| × 3% / 365
  5. 正現金: 無利息

每日(兩個季度之間):
  NAV = Σ(各股持股數 × 收盤價) + 現金 - 累計利息
```

## 風險與注意事項

1. **Yahoo calendarEvents 可用性**：需先驗證港股/日股的未來派息數據覆蓋率
2. **SEC EDGAR 爬蟲**：格式可能變動，需寫穩健的解析器
3. **13F 準確性**：45 天延遲、不含非美股/衍生品/放空，UI 必須明確標示
4. **高周轉型大師**：Citadel/Point72/Tiger/ARK 的模擬曲線會嚴重失真，必須加警告
5. **歷史股價依賴**：NAV 計算需要 3 年的每日股價，非美股可能數據不全
6. **Sandbox 限制**：SEC EDGAR 抓取需在 VPS 實機驗證
